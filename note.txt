ctrl+l - горячии клавиши очистки истории вывода команд терминала (аналог clear)
git remote -v
проверка привязки к удаленному репозиторию

git init
инициализация нового репозитория и создание скрытой папки .git

git add <имя файла / путь до файла>
добавить выбранный файл в индексацию

git add .
индексация всех файлов в репозитории

git commit -m <текст коммита>     
создать коммит (сохраненное изменение в коде) с сообщением

git log
показать историю коммитов (сохраненных изменений) если нужно (на mac / linux) выйти из истории коммитов в терминал нажмите q (quit - выход)

git branch -M main
переименование текущей ветки в main (важно, если ветка называется master)

git push -u origin main
пуш с установкой удаленной ветки по умолчанию (для того чтобы в будущем пользоваться короткой командой git push)

git remote remove origin
отвязать удаленный репозиторий

git pull origin <имя ветки>
забирает изменения с удаленной ветки
